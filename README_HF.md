# Beyond Space

Latent space navigation for RAVE audio models.

Upload 1–4 audio files, click a point on the board to set barycentric blend weights,
and generate a new sound by interpolating between the encoded latent trajectories.

Built with FastAPI + React/Vite. Model: [Intelligent Instruments Lab RAVE models](https://huggingface.co/Intelligent-Instruments-Lab/rave-models).
